# Study Tracker CLI

CLI-приложение на Python для помощи в учебе.

Этот проект был создан как практика Python, ООП, работы с JSON и Git/GitHub.

## Возможности

- Добавление предметов изучения
- Добавление цели изучения
- Добавление времени изучения в день
- Переход между двумя меню
- Просмотр всех предметов
- Возможность отметить выполнение предмета сегодня
- Удаление предмета
- Изменение всех параметров предмета
- Сохранение данных в JSON

## Что я практиковал в этом проекте

- Object-Oriented Programming (OOP)
- Работа с классами и объектами
- Работа с JSON файлами
- CRUD операции
- Обработка ошибок (`try/except`)
- Работа с циклами и условиями
- Структурирование проекта
- Git и GitHub
- Работа с CLI интерфейсом
